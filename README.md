# (C# 코딩) 파일 비교툴
## 개요
- C# 프로그래밍 학습
- 1줄 소개: 두 폴더의 파일들을 비교해서 상호 복사하는 툴
- 사용한 플랫폼:
	- C#, .NET Windows Forms, Visual Studio, GitHub
- 사용한 컨트롤:
	- Label, TextBox, ListBox, Button, SplitContainer, Panel, ListView
- 사용한 기술과 구현한 기능:
	- 컨트롤 배치 및 이벤트 처리
	- 폴더 선택 기능 (FolderBrowserDialog)
